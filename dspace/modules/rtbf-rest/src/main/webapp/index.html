Archimed RS API

  • GET / - Return this page.
  • GET /test - Return the string \"RS api is running\" for testing purposes.

Search

  • GET /search/sequences - Return list of sequences matching the query
  • POST /searchp/sequences - Return list of sequences matching the query
  • GET /search/episodes - (obsolete) Return list of episodes matching the query
  • POST /searchp/episodes - (obsolete) Return list of episodes matching the query
  • GET /search/series - (obsolete) Return list of series matching the query
  • POST /searchp/series - (obsolete) Return list of series matching the query
  • GET /group/episodes - Return list of episodes of assets matching the query
  • POST /groupp/episodes - Return list of episodes of assets matching the query
  • GET /group/series - Return list of series of assets matching the query
  • POST /groupp/series - Return list of series of assets matching the query

Handle (permanent id)

  • GET /handle/ARCHIMED/{handleid} - Returns the asset details identified by {handleid}
  • GET /handle/ARCHIMED/{handleid}/{owninghandleid} - Returns the sequence details identified by {handleid} in the context of the owning episode identified by {owninghandleid}
  • GET /handle/admin/clearCache - clear memory cache of handles, typically after a DB update

Asset

  • GET /sequences/{sequenceid} - Returns the sequence details identified by {sequenceid}
  • GET /sequences/{sequenceid}/metadata - Returns only metadata of the sequence identified by {sequenceid}
  • GET /episodes/{episodeid} - Returns the episode details identified by {episodeid}
  • GET /episodes/{episodeid}/metadata - Returns only metadata of the episode identified by {episodeid}
  • GET /episodes/{episodeid}/sequences - Returns list of sequences of the episode identified by {episodeid}
  • GET /series/{serieid} - Returns the serie details identified by {serieid}
  • GET /series/{serieid}/metadata - Returns only metadata of the serie identified by {serieid}
  • GET /series/{serieid}/episodes - Returns list of episodes of the serie identified by {serieid}
  • GET /series/top-series - Returns list of series representing the source repositories
  • GET /series/{topserieid}/series - Returns list of series of the source repository identified by {topserieid}

List Of Values

  • GET /contributors/names - Returns LOV for persons names
  • GET /contributors/roles - Returns LOV for persons roles
  • GET /contributors/{name}/roles - Returns list of roles for the person identified by {name}
  • GET /channels - Returns LOV for channels
  • GET /code_origines - (obsolete) Returns LOV for code_origines of support
  • GET /events - Returns LOV for events
  • GET /places/names - Returns LOV for places names
  • GET /places/roles - Returns LOV for places roles
  • GET /production_types - Returns LOV for production_type
  • GET /publishers - Returns LOV for producer
  • GET /royalty_codes - Returns LOV for royalty_code
  • GET /serie_titles - Returns LOV for series titles
  • GET /subjects - Returns LOV for matters

Suggest

  • GET /suggestQ?pt={terms} - Suggests popular queries that have been made on {terms}
